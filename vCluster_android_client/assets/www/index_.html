웹 사이트를 만들어 보세요.

Mac OS X에 있는 iWeb, Pages 및 많은 다른 응용 프로그램을 사용하면
자신만의 웹 사이트를 빠르고 쉽게
생성하여 발행할 수 있습니다.

Mac에서는 자신만의 웹 사이트를 손쉽게 생성하여 발행할 수 있습니다. 사이트만 준비된다면, 발행하는 일은 아주 쉽습니다.

시스템 환경설정을 열고 공유를 클릭한 다음 웹 공유를 선택하십시오.

그러면 끝난 것입니다. 이제 집 또는 직장에서 여러분의 개인 네트워크를 통해 자신의 웹 사이트를 이용할 수 있습니다.

인터넷에 연결되어 있다면 여러분의 친구들도 어디에서나 이 웹 사이트를 방문할 수 있습니다. 공유 환경설정에 있는 주소를 친구들에게 보내기만 하면 됩니다.

Apache 파워

웹 공유는 Mac OS X에 포함된 업계 표준 기술인 Apache 웹 서버를 기반으로 합니다. 웹 서버에 대한 추가 정보는, Apache 설명서를 참조하십시오.
